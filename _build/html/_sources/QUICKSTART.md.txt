## Using in Your Own Project

### 1. Copy Extension Files
```bash
cp _ext/custom_figure.py your-project/_ext/
cp _static/custom_figure.css your-project/_static/
```

### 2. Update conf.py
Add to your Sphinx configuration:
```python
import sys
from pathlib import Path

sys.path.insert(0, str(Path('_ext').resolve()))

extensions = [
    'myst_parser',
    'custom_figure',
]
```

### 3. Use in Documents
````markdown
```{figure} path/to/image.png
:author: Your Name
:license: CC-BY-SA
:date: 2025-01-15
:width: 500px

Caption text here.
```
````

## Valid Licenses

The extension recognizes 14 license types:
- CC0, CC-BY, CC-BY-SA, CC-BY-NC, CC-BY-NC-SA, CC-BY-ND, CC-BY-NC-ND
- MIT, Apache-2.0, GPL-3.0, BSD-3-Clause
- Public Domain, Proprietary, All Rights Reserved

## Customization

### Add New Licenses
Edit `_ext/custom_figure.py`:
```python
VALID_LICENSES = [
    'CC0',
    'CC-BY',
    'Your-Custom-License',  # Add here
]
```

### Change CSS Styling
Edit `_static/custom_figure.css` to customize appearance.

### Strict Mode
To make builds fail on missing licenses, modify the 
`check_all_figures_have_license()` function to raise exceptions.

## Troubleshooting

### Build Errors
```bash
# Reinstall dependencies
pip install -r requirements.txt --break-system-packages

# Clean and rebuild
make clean
make html
```

### Import Errors
Make sure `_ext` is in Python path in conf.py:
```python
sys.path.insert(0, str(Path('_ext').resolve()))
```

### CSS Not Loading
Verify in conf.py:
```python
html_static_path = ['_static']
html_css_files = ['custom_figure.css']
```
