## Best Practices

When using the custom figure directive:

1. **Always include a license** - Even if it's "All Rights Reserved"
2. **Use standard license identifiers** - Check the valid licenses list
3. **Provide author information** when known
4. **Include creation dates** for better record keeping
5. **Add descriptive alt text** for accessibility
6. **Write meaningful captions** that explain the figure

## Common Mistakes to Avoid

### ❌ Missing License

```markdown
<!-- This will generate a warning -->
```{figure} image.png
:author: John Doe
:date: 2025-01-15
```
```

### ❌ Invalid License

```markdown
<!-- "CUSTOM" is not a valid license -->
```{figure} image.png
:license: CUSTOM
```
```

### ❌ Wrong Date Format

```markdown
<!-- Use YYYY-MM-DD, not MM/DD/YYYY -->
```{figure} image.png
:license: MIT
:date: 01/15/2025
```
```

### ✅ Correct Usage

```markdown
```{figure} image.png
:author: John Doe
:license: CC-BY-SA
:date: 2025-01-15
:alt: Description
:width: 500px

Proper caption here.
```
```
